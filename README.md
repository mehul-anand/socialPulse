# Social Pulse
Social Media app project for neogCamp [ProjectTwo]

# Features
1. User Feed
2. Post Creation
3. Likes
4. Profile
5. Add Bio
6. Add Portfolio URL
7. Follow/Unfollow users
8. Explore Feed
9. Bookmark Posts
10. Authentication
11. Sign-up 
12. Login 
13. Logout

# Link[https://social-pulse.netlify.app/]

# Build with:
Mockbee (for backend)
React JS
React Router
React Icons
Tailwind CSS
Cloudinary
